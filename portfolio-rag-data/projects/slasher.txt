# Slasher

Slasher is an immersive, web-based VR-style rhythm game where you use your hands to slash incoming blocks to the beat. Powered by **MediaPipe** for hand tracking and **Three.js** for 3D visuals, it offers a unique, controller-free gaming experience directly in your browser.

## üéÆ Features

-   **Hand Tracking**: Real-time hand detection using your webcam (no VR headset required).
-   **Rhythm Gameplay**: Slash blue and pink blocks with your corresponding hands.
-   **Audio Reactive**: Visuals and lighting pulsate to the music.
-   **Dynamic Difficulty**: The game creates more challenging patterns and speeds up as you survive longer.
-   **High Score System**: Track your best runs.
-   **Immersive Visuals**: Neon aesthetic with post-processing effects.

## üïπÔ∏è Controls

1.  **Start Game**: Press the **Spacebar**.
2.  **Gameplay**:
    -   Use your **Left Hand** to control the **Pink Saber**.
    -   Use your **Right Hand** to control the **Blue Saber**.
    -   Slash blocks of the matching color to score points.
    -   Avoid missing blocks to save your lives (Start with **5 Lives**).

## üöÄ Installation & Setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/slasher.git
    cd slasher
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Visit `http://localhost:5173` (or the URL shown in your terminal). Allow camera access when prompted.
